Step 1: Initial Setup

Running for the first time? If yes, follow the instructions for your operating system. If not, skip this step.

Windows 10/11

  • Run the WinPcap.exe installer.
  • Ensure "Run driver on boot up" is checked.

macOS

  • Install Homebrew from https://brew.sh/.
  • Run the following command in Terminal:
    brew install libpcap libuv switch-lan-play

Debian/Ubuntu

  • Open Terminal in the Linux folder provided in this guide.
  • Run:
    sudo apt install libpcap0.8 libuv1
  • Then:
    chmod +x lan-play-Linux

Step 2: Running LAN-Play

Windows 10/11

  • Run lan-play.exe.

macOS

  • Run in Terminal:
    sudo lan-play

Debian/Ubuntu

  • Run in Terminal:
    sudo ./lan-play-Linux

Step 3: Selecting a Server

You can find servers here: http://lan-play.com/

Note: Ensure you and your friend use the same server.

Step 4: Configuring Your Switch

You need to manually edit the configuration of your local network with the following information:

  • IP Address: 10.13.XX.YY (Choose any from 10.13.0.1 to 10.13.255.254, except 10.13.37.1. Do not use the same IP as your friend.)
  • Subnet Mask: 255.255.0.0
  • Gateway: 10.13.37.1

Step 5: Resetting Network Settings

Done playing? On your Switch's network settings, click "Reset to Default".
