    • Step 1: First Time Setup

    Windows

    • Run WinPcap.exe and check "Run driver on boot up"

    macOS

    • Install Homebrew from brew.sh
    • brew install libpcap libuv switch-lan-play

    Linux (Debian/Ubuntu)

    • sudo apt install libpcap0.8 libuv1
    • chmod +x lan-play-linux

    Step 2: Running LAN-Play

    • Windows: run lan-play.exe
    • macOS: run sudo lan-play in terminal
    • Linux: run sudo ./lan-play-linux in terminal

    Step 3: Pick a Server

    Visit http://lan-play.com/ and choose the same server as your friend.

    Step 4: Configure Your Switch's Network Settings

    • IP Address: 10.13.XX.YY (Don't use 10.13.37.1, and don't use the same IP as your friend)
    • Subnet Mask: 255.255.0.0
    • Gateway: 10.13.37.1

    Are You Done Playing?

    On your Switch, reset your network settings to default.
